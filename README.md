# DT Money

Projeto desenvolvido a partir da plataforma da rockeseat no module de ReactJS de implementação e consumo de APIs + Performance.

## Tecnologias

- Typescript
- Vite
- Axios
- React Hook Form
- Zod
- Radix UI
- Styled Components

## Para rodar a aplicação + API Fake

```shell
pnpm run json:server
```

```shell
pnpm run dev
```
